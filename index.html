reTolkienddit

Technology
HTML
CSS
JS
React
Redux
API
Summary

reTolkienddit is a React web app, which utilizes Redux and React-Router. The app fetches most popular posts related to Tolkien or LOTR interests from Reddit API. The details of each post such as image content, vote, and comments are displayed for users to easily consume.

The app is mainly composed of three integrated features: Subscribed Tag/Account, Post, and Comments. Custom tags/accounts functionality can be developed, which will allow users to add their own personalized subscriptions, resulting in a more tailored experience.

One of the features of app is its Reddit-like mock-up vote functionality, which allows users to vote on posts. Additionally, the search functionality is optimized through its own routing system.

The design of the app is modern and elegant, with bright neutral colors and orange accents that match the theme of Reddit.

Your sweet ASF Diary

Technology
HTML
CSS
JS
Summary

Thought of as a personal blog with an astral-utopian-divinatory theme, Your Sweet ASF Diary also serves as a sandbox for experimenting with various stylistic and functional choices in CSS, HTML, and JS.

The website is designed as a grid with multiple boxes, where each box represents a specific feature such as a fairy corner for divination, a survey on futurism and utopianism, or give the opportunity to read stories and view pretty pictures ^^`. The most ambitious box is the 'Hangman Crystal Edition', which puts a magical Minecraft aesthetic twist on the popular game.

The website's layout features vibrant, warm, yet balanced colors. The design utilizes repeating gray cells of different sizes, providing a consistent and structured feel. The website leverages the external library Isotope to manage the layout, seamlessly adapting to different tile heights and adds filtering tags.

CPU Simulator

Technology
Python
Terminal
Summary

CPU Simulator is a Python-written processor design that consists of CPU, Memory, and ISA protocol. It includes three types of memory - Main Memory, Cache, and Register - with different access speeds, as well as Cache memory with LRU replacement and write-back policy. The simulation features a fetch, decode, execute cycle and reads data and command lists from an external .txt file.
